## 1. 스토어 등록정보 (Store Listing)

### **이름 (Title)**
- **국문**: PagePost - 웹 위의 스마트 메모 & 마크업
- **영문**: PagePost - Contextual Web Notes & Markup
- *제한: 최대 45자*

### **한 줄 설명 (Summary)**
- **국문**: 웹페이지 어디서나 메모와 마크업을 남기고, 모든 정보를 한곳에서 관리하세요.
- **영문**: Add sticky notes and markup directly on any webpage and manage them in one place.
- *제한: 최대 150자*

### **상세 설명 (Detailed Description)**
PagePost는 여러분의 웹 브라우징 경험을 생산적인 작업 공간으로 바꿔줍니다.
기존의 북마크나 텍스트 복사만으로는 부족했던 정보 수집, 이제 웹페이지 위에 직접 쓰고 그려보세요.

**주요 기능:**
- **📍 컨텍스트 메모**: 방문 중인 웹페이지의 특정 위치에 스티키 노트를 부착합니다. 나중에 다시 방문하면 그 자리에 메모가 그대로 남아있습니다.
- **🖌️ 강력한 마크업 도구**: 중요한 부분에 형광펜을 칠하거나, 펜 도구로 자유롭게 그림을 그리고 화살표로 강조하세요.
- **🖥️ 대시보드 & 캔버스 뷰**: 여러 도메인에 흩어진 메모들을 리스트 또는 캔버스 형태로 한눈에 관리하고 프로젝트별로 분류하세요.
- **🏷️ 스마트 태그 & 검색**: #태그를 사용하여 정보를 체계화하고 필요할 때 즉시 찾아내세요.
- **🔌 워크플로우 통합**: 작성한 메모를 Notion, Slack, Trello로 전송하여 팀원과 공유하거나 업무로 연결하세요.
- **🎨 개인화 설정**: 폰트, 크기, 테마 색상, 툴바 위치 등 사용자 맞춤형 UI를 제공합니다.
- **🔒 데이터 보안**: 모든 데이터는 브라우저 로컬 저장소에 안전하게 보관되며, 필요할 때 JSON으로 내보내고 가져올 수 있습니다.

PagePost와 함께라면 더 이상 중요한 아이디어를 놓치지 않습니다. 지금 바로 시작해보세요!

## 2. 개인정보 보호 (Privacy & Permissions)

### **단일 목적 설명 (Single-Purpose Description)**
웹페이지에 컨텍스트 기반의 메모와 마크업을 추가하고 관리하기 위한 확장 프로그램입니다.

### **권한 사용 목적 (Permission Justification)**
- **`storage`**: 사용자의 메모, 마크업 데이터 및 개인 설정을 브라우저에 저장하기 위해 필요합니다.
- **`activeTab`**: 현재 사용 중인 탭에 메모와 마크업을 표시하고 UI를 렌더링하기 위해 사용합니다.
- **`contextMenus`**: 마우스 우클릭을 통해 빠르게 메모를 생성하는 기능을 제공합니다.
- **`sidePanel`**: 메모 목록과 대시보드를 사이드바 공간에서 효율적으로 보여주기 위해 사용합니다.
- **`tabs` & `webNavigation`**: 사용자가 다른 페이지로 이동할 때 해당 URL에 맞는 데이터를 자동으로 불러오기 위해 필요합니다.
- **`<all_urls>` (Host Permissions)**: 모든 웹사이트에서 메모와 마크업 기능을 사용할 수 있도록 하기 위함입니다.

### **데이터 사용 세부 정보 (Data Usage)**
- **사용자 데이터 수집 여부**: 아니요 (No)
- **로컬 저장 방식**: 모든 데이터는 사용자의 로컬 저장소(`chrome.storage.local`)에만 저장되며, 외부 서버로 수집되지 않습니다.
- **예외**: 사용자가 직접 설정한 Notion, Slack, Trello 통합 기능을 사용할 때만 해당 서비스의 API를 통해 선택한 데이터가 전송됩니다.
